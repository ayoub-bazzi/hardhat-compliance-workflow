# Project Master Overview — HardHat Compliance

## 1. Product Summary & Mission
HardHat Compliance is an enterprise-grade automated risk management and compliance tracking system built specifically for General Contractors (GCs) managing subcontractors in the construction ecosystem. Its goal is to minimize liability, prevent fraud, and eradicate manual compliance verification data entry through real-time AI document parsing and multi-tier version verification.

## 2. Core Feature Matrix
* **The Bouncer Engine:** Document recognition and matching built on Gemini 2.5 Flash.
* **The Safe Vault:** Zero-clutter automated file versioning tracking active documents vs historical trail elements.
* **The Risk Register:** Dynamic fleet heatmap sorting subcontractors by risk matrix vectors (Expired, Mismatched, or Missing documentation).
* **Payment Gate Escrow:** Financial hold infrastructure tied directly to calculated database risk flags.

## 3. Production Philosophy
* **Extreme Friction Reduction:** GCs should never copy/paste metadata from a W9 or COI. The system processes fields instantly at the perimeter.
* **Legal Shielding:** The system favors keeping historical files intact in an immutable log while exposing only the active version in production spaces.
